# Premium Care Finance Operating System (Finance OS)

## Overview

The Premium Care Finance Operating System (Finance OS) is a board-level
financial management framework designed specifically for Premium Care.
It has been developed progressively through Versions 1--8 to evolve from
a simple budgeting workbook into an executive management, governance,
forecasting and reporting platform.

The purpose of the Finance OS is to provide a single source of truth for
the organisation's financial information, enabling Directors, Finance,
Operations, HR, Compliance and Senior Management to make informed
decisions using consistent data.

# Recommended Workflow

1.  Enter all transactions into the Transaction Hub.
2.  Record invoices, payroll and expenses.
3.  Update contracts and council income.
4.  Review Management Accounts monthly.
5.  Review KPIs weekly.
6.  Produce monthly Board Pack.
7.  Produce quarterly Investor Pack.
8.  Update forecasts every month.
